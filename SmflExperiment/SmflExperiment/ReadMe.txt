The programs starts with a prompt asking for a size for the grid. It then makes a grid of cells that react based on the following rules to step through and then display iterations. 

1) if a cell has 4, 5, or 6 neighbors, it dies or stays dead 
2) if a cell has 2 or 3 neighbors, it lives or becomes alive 
3) if a cell has 0 or 1 neighbors, it dies or stays dead
